# AI Resume Analyzer

An intelligent web application that analyzes resumes using AI to provide
insights, suggestions, and improvements. This tool helps users optimize
their resumes for better chances of selection by recruiters and ATS
(Applicant Tracking Systems).

------------------------------------------------------------------------

## Features

-   Upload Resume (PDF/DOCX)
-   AI-based Resume Analysis
-   Skill Extraction & Matching
-   Keyword Optimization Suggestions
-   Resume Score Generation
-   Improvement Recommendations
-   ATS Compatibility Check

------------------------------------------------------------------------

## Tech Stack

-   Frontend: HTML, CSS, JavaScript
-   Backend: Python / Node.js
-   AI/NLP: OpenAI API / NLP Libraries
-   Database: MongoDB / MySQL
